# Spring Data JPA - Ejercicios

Proyecto desarrollado como parte de la formación en **Spring Data JPA**, enfocado en el acceso a datos mediante JPA y Hibernate.

---

##  Funcionalidades implementadas

- Repositorios JPA
- Consultas personalizadas con `@Query`
- Métodos derivados de Spring Data
- Registro de logs con Logger
- Validaciones en entidades
- Manejo de excepciones personalizadas
- Ejecución de pruebas con datos de ejemplo
- Capturas de funcionamiento incluidas en el repositorio

---

## 🛠️ Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven

---

## ▶️ Ejecución del proyecto

1. Clonar el repositorio
2. Ejecutar la clase principal:
   `SpringDataJpaApplication`
3. Revisar la consola para ver:
   - Logs de ejecución
   - Consultas generadas por Hibernate
   - Resultados de las operaciones JPA

---

## 📸 Evidencias

Las capturas de ejecución se encuentran en la carpeta:
`capturas`

Incluyen:
- Consultas con `@Query`
- Métodos derivados
- Logs de ejecución
- SQL generado por Hibernate
